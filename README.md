# Cleaning-Website
CleanByYami is a production-ready cleaning service booking website developed for a real client. The platform enables customers to book residential and office cleaning services online while providing the business owner with an admin system to manage bookings and customer inquiries.
This project represents a real-world freelance/client deployment focused on usability, booking workflow, and business automation for a local service provider.
🚀 Features
Service listing with pricing (basic, deep, hourly cleaning)
Online appointment booking system
Contact and inquiry forms
Admin dashboard for managing:
Bookings
Customer messages
Service requests
Responsive design for mobile and desktop
Database integration for storing bookings
Email/notification handling for new bookings
